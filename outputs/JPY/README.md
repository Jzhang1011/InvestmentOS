# JPY Decision Brief Archive

Store reviewed decision briefs here using:

```text
YYYY-MM-DD-jpy-decision-brief.md
```

Example:

```text
2026-07-22-jpy-decision-brief.md
```

## Publication Status

Add metadata to each brief:

```yaml
status: draft | approved | published
reviewed_by:
reviewed_date:
sources_verified: true | false
```

Do not publish AI-generated analysis without source and factual review.
