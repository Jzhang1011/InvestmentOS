## Escalation Rules

- No important indicator changed → routine monitoring.
- One important indicator changed → watch.
- Multiple aligned indicators changed → investigate.
- Core assumption changed → reassess thesis.
- Several assumptions changed → review portfolio exposure.

## Review Discipline

- Use primary data when available.
- Compare with market expectations.
- Separate temporary shocks from structural changes.
- Do not react to one release in isolation.
